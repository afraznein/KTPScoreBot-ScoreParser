# KTPScoreBot-ScoreParser
KTP Score Bot - Score Parser

## Features
- **Automatic Score Parsing**: Parses Discord messages and updates Google Sheets
- **Weekly Banner Posting**: Posts weekly match banners to Discord (Monday 8 AM)
- **BYE Match Auto-Scoring**: Automatically awards average points to teams with BYE weeks (Daily 8 AM)
- **Division Support**: Bronze, Silver, Gold divisions
- **Idempotent Processing**: Safe to re-run without duplicating scores
- **Full Audit Trail**: Receipts logged to _ScoreReceipts sheet
